# VMU Training

A tablet-focused intermodal yard training simulator for hostler operators. The application recreates the VMU workflow with work-order selection, equipment filtering, chassis matching, sequencing alerts, and guided container-mating scenarios.

## Local development

Requires Node.js `>=22.13.0`.

```bash
npm install
npm run dev
```

Open `http://localhost:3001/`.

## Build

```bash
npm run build
```
